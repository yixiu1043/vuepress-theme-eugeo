## 安装依赖
```sh
yarn add stylelint stylelint-config-standard stylelint-order stylelint-scss --dev
```

### `.stylelintignore`文件
忽略不需要检查的目录
```
build
config
static
dist
```

## 脚本命令
```json
"scripts": {
  "stylelint": "stylelint src/*.{html,vue,css,sass,scss} --cache --fix"
}
```
